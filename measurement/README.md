# Off-Page Authority Measurement

Track the plan weekly in `off-page-authority-tracker.csv`.

## What to measure

- `referring_domains_total`: total referring domains from your backlink tool of choice
- `new_relevant_referring_domains`: new domains that actually overlap with SEO, SaaS, localization, or global growth
- `brand_search_impressions`: Search Console impressions for branded queries like `seo with faiz`, `faiz seo`, and close variants
- `brand_search_clicks`: Search Console clicks for those branded queries
- `international_seo_cluster_impressions`: Search Console impressions for the target service cluster
- `international_seo_cluster_clicks`: Search Console clicks for the same cluster
- `qualified_referral_sessions`: referral traffic from mentions, podcasts, directories, and partner pages that stays engaged
- `qualified_referral_leads`: lead submissions or booked calls tied to those referral sources
- `mentions_earned`: count of live editorial mentions or interviews
- `podcasts_booked`: count of confirmed or published guest appearances
- `directories_live`: count of directory profiles that are live and accurate

## Query cluster to watch

- `international seo consultant`
- `international seo expert`
- `international seo services`
- `international seo for us companies`
- brand queries for `SEO With Faiz` and `Faiz`

## Weekly routine

1. Update the CSV every Monday.
2. Record only relevant domains, not vanity link gains.
3. Add short notes on what changed: mentions, podcasts, directory approvals, or big query shifts.
4. If a channel produces no replies for three straight weeks, reduce effort and move resources.
