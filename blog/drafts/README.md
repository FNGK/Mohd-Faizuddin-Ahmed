# Blog Drafts Folder

This folder stores auto-generated blog drafts that require manual review.

## Rules

- Keep `approved: false` until editorial review is complete.
- Validate factual claims and citations before approval.
- Ensure at least:
  - 2 authoritative external citations
  - 3 internal links
  - one clear CTA to `contact/index.html`

After approval, move draft content into `blog/posts/` using the publish validator script.
