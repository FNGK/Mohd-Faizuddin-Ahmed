Growth Marketing Case Study

Hyderabad Globe FC digital transformation

This case study shows how I rebuilt a legacy site to improve speed, local SEO structure, and content readiness for search engines and answer engines.

Context

The previous site ran on Wix and had limited flexibility for performance and structured SEO improvements.

Problem

  • Slow page performance.
  • Weak local SEO structure for league and coaching searches.
  • Limited schema coverage for clearer search understanding.

Approach

  • Rebuilt the website with React and Tailwind.
  • Implemented local SEO page architecture and metadata alignment.
  • Added JSON-LD schema signals for stronger semantic clarity.

Results

  • 95+ Lighthouse performance and accessibility benchmarks.
  • Faster loading experience across key pages.
  • Clearer local intent structure for search visibility work.

Tools used

React, Tailwind CSS, Lighthouse, JSON-LD, local SEO planning.
