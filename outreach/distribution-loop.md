# Distribution Loop

Use this after every earned mention, podcast appearance, directory approval, or guest contribution.

## Site update checklist

1. Add the new mention to `mentions.html`.
2. Add one internal link from the most relevant proof or resource page to `services/international-seo.html`.
3. If the mention references a framework or checklist, add that mention link to the relevant resource page.
4. Update the proof summary on the homepage if the earned mention materially strengthens authority.

## Short proof update template for site

Title: `Mention: {{publication_or_podcast}}`

Fields to log:
- Date
- Channel type
- Link
- Topic discussed
- Asset referenced
- Follow-on internal links added

## Internal-link targets

Prioritize links into:

- `services/international-seo.html`
- `resources/international-seo-diagnostic-framework.html`
- `resources/hreflang-market-rollout-checklist.html`
- `resources/international-seo-failure-patterns-us-companies.html`
- `resources/international-seo-expert-media-kit.html`

## Repurposing sequence

1. Publish or log the mention on the site.
2. Post the concise LinkedIn version.
3. Add the mention to outreach replies as fresh proof.
4. Link the mention from the most relevant resource page.
5. Record the result in the measurement tracker.
