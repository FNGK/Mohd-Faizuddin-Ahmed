# 15-Minute Diagnostic Call Script

Goal: convert qualified calls into paid starter engagement.

## Minute 0-2: Set context

- Thank them for time.
- Confirm call goal: identify top blockers and next action.
- Ask permission to be direct and practical.

## Minute 2-6: Diagnose

Ask:
- What is your primary growth goal for the next 90 days?
- What channels currently bring leads?
- What result is currently underperforming (traffic, leads, bookings, sales)?
- What has already been tried?

## Minute 6-10: Frame the problem

Use this structure:
- "From what I reviewed, the biggest bottleneck is {{CORE_BOTTLENECK}}."
- "This usually impacts {{BUSINESS_OUTCOME}}."
- "The fastest path is: fix A, then B, then C."

## Minute 10-13: Offer

Starter close:
- Offer: SEO Audit or Local SEO Sprint.
- Scope: exact inclusions, exact timeline, exact deliverable.
- Price: INR 3000 or USD 100, upfront.

Backend setup (mention softly):
- "If you want me to implement after the audit, we can discuss a monthly retainer."

## Minute 13-15: Close

Direct close line:
- "If this aligns, I can start as soon as payment is confirmed today."

If yes:
- Send payment link and confirmation message immediately.
- Confirm kickoff date and required inputs.

If maybe:
- Ask one objection question: "What is stopping the decision right now?"
- Resolve one objection only, then ask for decision.

If no:
- Thank them and ask permission to reconnect in 30 days.

## Post-Call Admin (Mandatory)

- Update CRM fields: Call_Completed, Outcome, Objection, Next_Action.
- If closed: mark `Closed_Deal = Yes`, record value, start delivery checklist.
- If not closed: set next follow-up date before ending the session.
