# Payment and Onboarding SOP

## Payment Flow

Use this immediate sequence after a verbal "yes" on call:

1. Send payment link within 5 minutes.
2. Send scope summary in writing.
3. Confirm kickoff date only after payment confirmation.

## Scope Confirmation Template

Subject: Confirmed scope - {{PRIMARY_OFFER}} - {{BUSINESS_NAME}}

Body:
- Offer: {{PRIMARY_OFFER}}
- Included: {{INCLUSIONS}}
- Excluded: implementation beyond scope
- Delivery timeline: {{TIMELINE}}
- Price: {{PRICE}}
- Payment: upfront

Please reply with "Confirmed" so I can start.

## Kickoff Message Template

Hi {{CONTACT_NAME}},

Payment received. Thank you.

To begin, please share:
- primary goal for the next 90 days
- top 3 pages to prioritize
- access details (if applicable)

Kickoff date: {{DATE}}
First delivery date: {{DATE}}

Regards,  
Faiz

## Onboarding Checklist

- [ ] Payment received and logged in CRM.
- [ ] Scope confirmation sent.
- [ ] Inputs collected from client.
- [ ] Delivery deadline added to calendar.
- [ ] Follow-up Q&A slot pre-booked.
