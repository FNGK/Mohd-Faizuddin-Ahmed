# Button Eyes Resort SEO Quotation

Prepared for: Button Eyes Resort  
Prepared by: SEO With Faiz  
Date: 2026-05-25

## Recommended Proposal

### 90-Day SEO Recovery and Growth Foundation

**Investment:** `INR 1,50,000 + GST`  
**Billing:** `INR 50,000 + GST per month for 3 months`

This is the package that should be presented first. A cheaper quote is possible, but it risks leaving the work half-done because this site needs both cleanup and structured follow-through.

This pricing is intentionally kept friendly for the first 90 days so the engagement can start with momentum, show visible progress, and build into a longer-term growth relationship if the recovery phase performs as expected.

## What Is Included

### Month 1: Technical Recovery

- Sitemap fix and submission validation
- crawl and indexation review
- duplicate blog URL cleanup plan
- canonical, redirect, and noindex recommendations where needed
- fix list for visible shortcode/template leftovers
- metadata rewrite plan for priority pages
- GA4 and GSC baseline review

### Month 2: Commercial Page Cleanup

- homepage SEO rewrite direction
- FAQ rewrite framework with real customer questions
- experiences page rewrite framework
- booking page structure recommendations
- keyword mapping for core commercial pages
- internal linking recommendations across stay, amenities, bookings, contact, and experience pages

### Month 3: Local Growth Foundation

- local intent keyword refinement
- Google Business Profile optimization recommendations
- content calendar for resort-focused topics
- reporting dashboard and KPI review
- implementation QA and re-audit of completed fixes

## Deliverables

- one prioritized SEO roadmap
- one technical fix tracker
- metadata recommendations for up to `15` priority URLs
- content recommendations or rewrite briefs for up to `5` priority pages
- duplicate content cleanup plan for blog/archive URLs
- one 90-day keyword and content map
- monthly progress summary
- two review calls per month

## Assumptions Behind This Quote

This quotation assumes:

- WordPress/admin access will be provided
- one decision-maker or point of contact will approve changes
- implementation support is available on the website side
- the current engagement is audit + recovery planning + SEO execution guidance
- this is not a full redesign or custom booking-engine build

## Exclusions

Not included in the above fee unless separately approved:

- full website redesign
- custom development outside agreed SEO fixes
- plugin or third-party license costs
- paid ads or media buying
- backlink buying or PR placement fees
- photography, videography, or brand design
- writing large volumes of net-new blogs beyond the scoped pages
